Блок навигации

  • Ширина 15em
  • Высота определяется контентом
  • Должен присутствовать цвет фона
  • Распологается слева
  • Блок навигации наполните контентом достаточным, чтобы он по высоте начинал захватывать вторую статью, как показано в этом примере

Статья 1

Размер блока: Занимает всю доступную ширину
Высота по размеру контента
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу .2em
