Блок навигации
  • Ширина 15em
  • Высота определяется контентом
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере
Статья 1
Размер блока:
Занимает всю доступную ширину
Высота по размеру контента
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей .2em, при помощи свойста margin-bottom
Статья 2
Размер блока:
Занимает всю доступную ширину
Высота по размеру контента
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей .2em, при помощи свойста margin-bottom. Картинка выставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки.
Далее идёт ещё какой-то текст просто для демонтрации обтекания картинки.
Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внутрь контентом.
